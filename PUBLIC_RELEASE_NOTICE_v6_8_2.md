# Public release notice (v6.8.2 JCIS compliance)

This is the JCIS-compliance public-minimised release of N1-MHKit prepared for Journal of Computer Information Systems review and citation. It aligns the public repository package with the v6.8.2 JCIS manuscript package while preserving the public release boundary, public tables, figure sources, validation scripts and non-clinical interpretation boundary.

This release includes public-minimised processed tables, public-specific metadata, documentation, reproducible figure-source files, validation scripts and publication-ready PDF/SVG/PNG/TIFF figure exports.

Source-layer files, source media objects, high-resolution temporal details, source filenames, contextual text, raw ECG PDF reports, PCG/WAV audio files and detailed participant-context materials are excluded from this public release. It is non-clinical and must not be used for diagnosis, treatment guidance, device certification, causal inference or population-level health conclusions.

No generative AI image model was used to create, generate or alter the figures or artwork.
