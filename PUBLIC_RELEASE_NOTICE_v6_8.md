# Public release notice (v6.8 final clean)

This is the final-clean public-minimised release of N1-MHKit prepared for Journal of Computer Information Systems review and citation. It includes public-minimised processed tables, public-specific metadata, documentation, reproducible figure-source files and validation scripts.

This release excludes source-layer files, source media objects, high-resolution temporal details, source filenames, contextual text and detailed participant-context materials. It is non-clinical and must not be used for diagnosis, treatment guidance, device certification, causal inference or population-level health conclusions.

No generative AI image model was used to create, generate or alter the figures or artwork.
