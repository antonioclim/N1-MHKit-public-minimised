# Release consistency statement (v6.8 final clean)

The GitHub repository tree, GitHub release asset and Zenodo deposit should contain the same public-minimised release contents, except for platform-generated GitHub source archives. The public release excludes source-layer files, source media objects, exact local timestamps, source filenames, contextual text and detailed participant context.

After publishing the Zenodo v6.8 version, cite the version-specific DOI shown on the Zenodo landing page in the article and submission-system fields. The package also records the Zenodo all-version concept DOI for stable repository discovery.
