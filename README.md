# N1-MHKit public-minimised release (v6.6)

N1-MHKit is a reproducible data-management artifact for multimodal personal health monitoring information systems. This author-confirmed public-minimised release supports inspection of the Journal of Computer Information Systems submission by providing code, metadata, documentation and processed public-minimised tables for artifact review.

## Scope

This public package is deliberately smaller than the restricted verification package. It is intended for research-software inspection, data-management review, reproducibility checking of public-minimised artefacts, and documentation of the Design Science Research artifact.

## Included

- Public-minimised processed tables.
- Public-minimised data dictionary.
- Design Science Research traceability metadata.
- Public-release gate documentation.
- Neutral architecture figures.
- Public package smoke-test and integrity-audit scripts.

## Excluded

Withheld source-layer files, source media objects, high-resolution temporal details, withheld source filenames, participant context and contextual text are not part of this public package. These materials are outside the public release boundary.

## Interpretation boundary

The demonstration package is non-clinical. It does not support diagnosis, treatment guidance, device certification, causal inference, or population-level health conclusions. The contribution is the curation artifact and its documentation.
