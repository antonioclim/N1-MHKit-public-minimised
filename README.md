# N1-MHKit public-minimised release (v6.8 final clean)

N1-MHKit is a reproducible data-management artifact for multimodal personal health monitoring information systems. This final-clean public-minimised release supports inspection of the Journal of Computer Information Systems submission by providing code, metadata, documentation, public-minimised processed tables and reproducible manuscript figure sources.

## Scope

This public package is deliberately smaller than the restricted verification package. It is intended for research-software inspection, data-management review, reproducibility checking of public-minimised artefacts, and documentation of the Design Science Research artifact.

## Included

- Public-minimised processed tables.
- Public-minimised data dictionary.
- Public-specific Design Science Research traceability metadata.
- Public-release gate documentation.
- Reproducible figure sources for manuscript Figure 1, Figure 2 and Supplementary Figure S1.
- PDF, SVG and high-resolution PNG figure exports.
- Public package smoke-test, schema-check and integrity-audit scripts.

## Excluded

Withheld source-layer files, source media objects, high-resolution temporal details, source filenames, participant context and contextual text are not part of this public package. These materials are outside the public release boundary.

## Figure provenance

The main manuscript figures are generated by deterministic Python scripts from CSV/JSON figure-source files. Supplementary Figure S1 is generated from public-minimised paired values. No generative AI image tool was used to create, generate or alter any figure or artwork.

To regenerate all manuscript figures, run:

```bash
python code/figures/generate_all_manuscript_figures.py
```

## Validation

To run the public-package checks:

```bash
python code/public_package_smoke_test.py
python code/validate_public_release.py --root .
python code/figures/generate_all_manuscript_figures.py
```

## Interpretation boundary

The demonstration package is non-clinical. It does not support diagnosis, treatment guidance, device certification, causal inference, or population-level health conclusions. The contribution is the curation artifact and its documentation.
